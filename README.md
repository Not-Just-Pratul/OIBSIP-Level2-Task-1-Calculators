# 🧮 Calculator Hub

A web-based calculator application featuring both **Basic** and **Scientific** calculators. Perfect for students, professionals, and anyone who needs quick calculations on the go.

## 🌐 Live Demo

Check out the live demo here:
- https://basic-calculator-example.netlify.app/

---

## 📋 Features

### **Basic Calculator**
- ✅ Addition, Subtraction, Multiplication, Division
- ✅ Percentage calculations
- ✅ Clear and Delete functions
- ✅ Clean, user-friendly interface

### **Scientific Calculator**
- ✅ All basic calculator functions
- ✅ Trigonometric functions (sin, cos, tan)
- ✅ Factorial calculations
- ✅ Advanced mathematical operations
- ✅ Parentheses support
- ✅ Professional styling

## 📱 How to Deploy on Netlify

### **Method 1: Drag & Drop (Easiest)**
1. Go to [netlify.com](https://netlify.com)
2. Sign up or log in
3. Drag and drop the entire `OIBSIP-Level2-Task1` folder onto Netlify
4. Your site is now live! 🎉

### **Method 2: GitHub Integration (Recommended)**
1. Push your code to GitHub
2. Go to [netlify.com](https://netlify.com) → "New site from Git"
3. Connect your GitHub account
4. Select your repository
5. Set **Publish directory** to: `Task 4/OIBSIP-Level2-Task1`
6. Click Deploy

### **Method 3: CLI**
```bash
npm install -g netlify-cli
netlify login
cd "Task 4/OIBSIP-Level2-Task1"
netlify deploy --prod
```
